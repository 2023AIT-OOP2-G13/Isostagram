# Isostagram


## アプリ概要

- 画像投稿簡易型SNSサイト
- ホーム画面にて投稿記事の一覧と投稿画面への移動。
- 投稿画面にてファイルを投稿タイトルを指定して投稿。
- ホーム画面にて投稿記事をクリックすると詳細画面に移動。
- 詳細画面にて記事のコメントを表示し、氏名とコメントをそのまま記入できる。
- また、詳細画面から投稿も可能。
- 詳細画面と投稿画面から、ホームに移動可能。

## アプリ完成イメージ

- トップページイメージ
- 投稿ページイメージ
- 投稿の詳細ページイメージ
